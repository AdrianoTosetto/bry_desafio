
# Teste técnico

## Como rodar


criar a pasta build na raiz do projeto caso não exista:
```bash
  mkdir build
```
Em seguida entrar na pasta

```bash
  cd build
```
Rodar o CMake para gerar os arquivos de build
```bash
  mkdir build
```
Executar o make para geração do build e rodar

```bash
  make && ./desafio_bry
```
